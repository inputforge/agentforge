# AgentForge Docs

AgentForge is a Kanban board that spawns AI coding agents in isolated git worktrees — one agent per ticket.

- [Workflow](./workflow.md) — Tickets, columns, agents, review, and merge
- [Agents](./agents.md) — Supported agents, prompting tips, and how worktrees work
- [Configuration](./configuration.md) — Repository setup, environment variables, and data storage
